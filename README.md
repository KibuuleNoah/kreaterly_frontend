# Kreaterly Frontend

A modern, responsive frontend application built with React and TypeScript for the Kreaterly platform.

## Features

- ⚡ Built with React and TypeScript for type safety
- 🎨 Responsive design for all device sizes
- 🚀 Modern development workflow with optimized build process
- 🔧 Well-structured component architecture

## Tech Stack

- **React** - UI library
- **TypeScript** - Type-safe JavaScript
- **Modern CSS** - Styling

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/KibuuleNoah/kreaterly_frontend.git
cd kreaterly_frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will open in your browser at `http://localhost:3000`

## Available Scripts

- `npm start` - Start the development server
- `npm run build` - Create a production build
- `npm test` - Run tests
- `npm run eject` - Eject configuration (one-way operation)

## Project Structure

```
src/
```
