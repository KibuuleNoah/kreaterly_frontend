# KreateRly Frontend Documentation

## Project Overview
KreateRly is a comprehensive platform designed to streamline the user experience in creating and managing digital content. The frontend is developed using modern web technologies to ensure a responsive and user-friendly interface.

## Tech Stack
- **React** - For building user interfaces.
- **Redux** - For state management.
- **Axios** - For making HTTP requests.
- **Sass** - For styling components.
- **React Router** - For navigation between different components.

## Authentication Methods
KreateRly supports multiple authentication methods:
- **Email and Password**: Users can create an account using their email and a secure password.
- **OAuth**: Integration with social login providers for a seamless sign-in experience (e.g., Google, Facebook).

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/KibuuleNoah/kreaterly_frontend.git
   ```
2. Navigate to the project directory:
   ```bash
   cd kreaterly_frontend
   ```
3. Install the required packages:
   ```bash
   npm install
   ```

## Setup Procedures
1. Create a `.env` file in the root directory and add the necessary environment variables (refer to `.env.example`).
2. Start the development server:
   ```bash
   npm start
   ```
3. Open your browser and navigate to `http://localhost:3000` to view the application.

## Missing Features
- **File Upload**: Currently, users cannot upload files directly through the app.
- **Dark Mode**: A dark mode feature is yet to be implemented, pending user feedback.
- **User Analytics Dashboard**: This feature is planned for future releases, allowing users to view performance metrics.
